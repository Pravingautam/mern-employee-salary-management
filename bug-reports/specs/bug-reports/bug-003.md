# Bug Report 003

## Title
Admin can delete employee who has active
salary records for current month

## Severity
High

## Who Gets Hurt
Payroll operator runs salary report and
that worker is missing from the list.
Worker does not get paid that month.

## Steps to Reproduce
1. Login as Admin
2. Go to Data Pegawai
3. Select any employee who has salary
   data in Data Gaji for current month
4. Click delete icon
5. Confirm deletion
6. Go to Data Gaji and run report

## Expected Result
System should warn admin that this employee
has active salary records and block deletion

## Actual Result
Employee gets deleted without any warning
They disappear from payslip report

## Root Cause
No validation check before deletion to
verify if employee has linked salary or
attendance records

## Impact
Worker loses their payslip for the month
Payroll operator has to manually fix records
